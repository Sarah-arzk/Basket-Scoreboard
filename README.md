#  Basket Scoreboard

A simple and modern basketball scoreboard built with HTML, CSS and JavaScript.

##  Features
- Add +1, +2, +3 points
- Automatic leader highlight
- Digital scoreboard design
- Reset with "New Game" button
- Clean responsive layout

## Technologies
- HTML
- CSS
- JavaScript

##  How to Run
1. Download the files
2. Open `index.html` in your browser
